# How to setup to local

1. Install nodejs
2. clone repo `git clone git@github.com:Stectech/GPS.git` for ssh or `git clone https://github.com/Stectech/GPS.git` for https
3. npm install 
4. bower install
5. on the terminal run `gulp`
